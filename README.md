# Mall Customers Clustering

## 📌 Description
This project uses **K-Means clustering** on the Mall Customers dataset to segment shoppers based on **Annual Income** and **Spending Score**. Visualizations (boxplots, pairplots, scatterplots) help explore distributions and cluster patterns. The elbow method is applied to determine the optimal number of clusters.

## 📂 Project Workflow
1. Load and explore dataset
2. Data visualization (Seaborn, Matplotlib)
3. Feature extraction for clustering
4. Elbow method for optimal k
5. K-Means clustering (6 clusters)
6. Visualization of clusters with centroids

## 🛠️ Libraries Used
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  
- Scikit-learn  

## 📊 Results
- Identified 6 distinct customer segments  
- Visualized clusters with centroids for better interpretation  
- Useful for targeted marketing and customer segmentation strategies
